Conjugation Practice

A project that focuses on helping users practice Japanese verb conjugations through a web application. This is the one I am most excited about. I've been enjoying learning Japanese since it's pretty challenging and a lot of fun. I wanted to make a program that would help me with the conjugation since it's pretty straightforward in most languages. Instead of using flashcards trying to memorize all the conjugations, I wanted to put them into action and see if I can remember the various inflections and the verb stem variations.

I really enjoyed making this one especially since I learned a lot about how web requests work and how they're much more involved than I realised.
